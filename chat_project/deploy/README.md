# 校园聊天系统部署说明

## 1. 交付内容

本部署包包含：

- `bin/chat_server.exe`：聊天服务端
- `bin/chat_web_gateway.exe`：网页访问网关
- `web_client/`：网页前端资源
- `config/server.conf`：系统配置文件
- `database/`：SQLite 数据库目录
- `logs/`：日志目录
- `start.bat`：一键启动
- `stop.bat`：一键停止

## 2. 运行环境

- Windows 10 / Windows 11 64 位
- 无需额外安装 Python
- 无需额外安装 Qt

本交付包已经包含运行所需的 Qt 依赖库。

## 3. 启动方法

直接双击：

```text
start.bat
```

系统将自动完成：

1. 启动聊天服务端
2. 启动网页网关
3. 自动打开浏览器

默认访问地址：

```text
http://127.0.0.1:8080
```

## 4. 停止方法

双击：

```text
stop.bat
```

## 5. 默认端口

- 聊天服务端 TCP：`8888`
- 网页网关 WebSocket：`9999`
- 网页访问地址 HTTP：`8080`

如需调整，可修改：

```text
config/server.conf
```

## 6. 数据与日志

- 数据库文件：`database/chat.db`
- 服务端日志：`logs/server.log`

消息会保存在 SQLite 数据库中，重新登录后可继续查看历史聊天记录。

## 7. 使用说明

1. 启动系统后打开浏览器页面
2. 注册账号
3. 登录账号
4. 查看在线用户
5. 点击联系人开始聊天

可同时打开多个浏览器窗口或无痕窗口，使用不同账号进行聊天测试。

## 8. 注意事项

- 首次启动时如果 `database`、`logs` 等目录不存在，程序会自动创建
- 如果 8080、8888、9999 端口被占用，请先关闭占用程序或修改配置文件
- 建议在本机或局域网环境中使用
